# [PL] HW1

> ### 최대공약수(GCD) 구하기
- 사용자가 입력한 값들의 최대공약수를 구하여 화면에 출력한다.
<br><br>

> ### 소수(prime number) 구하기
- 사용자에게 입력 받은 값들을 버블정렬을 통해 정렬한 후
- 각 값들 사이를 범위로 지정하여(ex. 1, 3, 8, 9 => 1~3, 3~8, 8~9)
- 범위 내에 존재하는 소수의 개수를 구하여 출력한다.
<br><br>

> ### 프로그램 수행 시간 구하기
- 사용자에게 값을 입력받은 순간부터 모든 결과 값이 출력되는 순간까지의 시간을 구하여 화면에 출력한다.
